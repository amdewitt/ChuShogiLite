Standard Sandbox Mode Instance

This instance uses sandbox mode (default), which shows all controls including Undo, New Game, Reset buttons, Edit tab, and Import section.

ViewOnly Mode Instance

This instance uses viewOnly mode, which hides game-changing controls (Undo, New Game buttons, Edit tab, Game Import section within Export/Import tab), disables board clicks for piece movement, and restricts certain settings, useful for viewing recorded games. Reset button, navigation controls, and Export functionality remain available. Note: allowIllegalMoves is set to true to test that viewOnly mode now respects the configured value.

FixedStart Mode Instance

This instance uses fixedStart mode (default), which blocks any settings that change the starting position.

FixedRules Mode Instance

This instance uses fixedRules mode, which blocks rule settings changes but allows editing and importing.

FixedStartAndRules Mode Instance

This instance uses fixedStartAndRules mode, which combines both fixedStart and fixedRules restrictions - hides Edit tab, restricts imports, and blocks rule changes.

Puzzle Mode Instance

This instance uses puzzle mode, which validates moves against a solution sequence. Rules settings and imports are blocked. Opponent moves execute automatically.
